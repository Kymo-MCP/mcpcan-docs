# MCP CAN 社区版

## 部署社区版

MCP CAN 社区版即开源版本，你可以通过以下方式部署 MCP CAN 社区版：

- [Kubernetes 部署](https://kymo-mcp.github.io/mcpcan-deploy/)<br>

​ 在 GitHub 上查看 [MCP CAN 社区版](https://github.com/Kymo-MCP/mcpcan) 开源代码。<br>

## 贡献代码

为了确保正确审查，所有代码贡献 - 包括来自具有直接提交更改权限的贡献者 - 都必须提交 PR 请求并在合并分支之前得到核心开发人员的批准。<br>

我们欢迎所有人提交 PR！如果你愿意提供帮助，可以在 [贡献指南](https://www.mcpcan.com/zh/guide/contribution) 中了解有关如何为项目做出贡献的更多信息。
